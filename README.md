# onion-scan-binaries
Onionscan precompiled binaries [s-rah/onionscan](https://github.com/s-rah/onionscan)

# Build Instructions
* clone [s-rah/onionscan](https://github.com/s-rah/onionscan)
  ```shell
  git clone https://github.com/s-rah/onionscan
  ```
* copy `Makefile` to folder.
* go to folder
* run command 
  ```shell
  make all
  ```
